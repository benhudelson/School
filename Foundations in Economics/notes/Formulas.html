Consumer Surplus = Value to buyers - Amount paid by buyers

Producer surplus = Amount received by sellers - cost to sellers

Total Surplus = Value to buyers - cost to sellers

Profit = Total Revenue - Total Cost

Average Total Cost = Total Cost / Quantity

Marginal Cost = △TC / △Q

Total Revenue = P X Q

Average Revenue = TR / Q

Marginal Revenue = △TR / △Q

Marginal Cost = △TC / △Q

Change in profit = Marginal Revenue - Marginal Cost = △TR / △Q - △TC / △Q = (△TR- △TC) / △Q

For a firm operating in a competitive market, profit maximizing output is produced when MR = MC.

The efficient scale for a firm occurs when ATC = MC.

Profit = TR - TC = (TR/Q - TC/Q) * Q = (P - ATC) * Q

If P < AVC, then a firm should shut down (short term)

If P < ATC, then a firm should exit (long term)
